# Python数据预处理

## 链家新房数据爬取预处理

### 目的

* 通过爬虫爬取链家的新房数据，并进行预处理。
* 输出格式：csv文件，包含：
	* 名称
	* 地理位置（行政区、街道、地址3个字段分别存储）
	* 房型（只保留最小房型）
	* 面积（按照最小值，整数）
	* 均价（元，整数）
	* 总价（万元，保留小数点后4位）

### 环境

* Windows10 Pro 2004，macOS 10.15。
* PyCharm，Python3.7
